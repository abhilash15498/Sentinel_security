Multi-Layer Detection

🎥

Face Recognition

Real-time detection using OpenCV Haar Cascade + DeepFace Facenet512. Burst enrollment captures 15 frames for robust matching.

🔍

Intrusion Detection

Monitors keyboard, mouse activity, and processes. Detects suspicious applications and unauthorized access patterns.

💾

USB Monitoring

Real-time USB device insertion/removal detection. Linux (udev events), Windows/Mac (partition polling).

🖥️

Screen Watching

Detects significant screen content changes. Captures evidence on suspicious activity with screenshot comparison.

🌐

Network Surveillance

Monitors active connections and detects suspicious ports. Baseline comparison for new connection alerts.

📧

Multi-Channel Alerts

Email, Telegram, desktop notifications, audio alarms, and TTS. Screenshot attachments for evidence.
